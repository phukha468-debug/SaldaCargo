## Правила для seed-скрипта

При парсинге этого файла скрипт должен:

1. Строки с `max_user_id = TODO` — вставлять пользователя с `max_user_id = NULL` и `is_active = false`. Пользователь не сможет войти в систему пока не привязан к MAX.
2. Строки с `phone = TODO` — вставлять `NULL`.
3. Если у одной строки несколько ролей (`extra_roles`) — создавать несколько записей в `user_roles`.
4. `assigned_reg` → JOIN с таблицей `assets` по полю `reg_number` для получения `asset_id`.
