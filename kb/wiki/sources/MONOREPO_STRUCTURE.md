# MONOREPO_STRUCTURE (Структура монорепозитория)
**Тип:** техническая архитектура
**Дата добавления:** 2026-04-30
**Источник:** [[raw/01_foundations/MONOREPO_STRUCTURE.md]]

## Ключевые идеи
- **Фреймворк:** Использование `pnpm workspaces` для управления монорепозиторием.
- **Разделение приложений:** `apps/web` (дашборд) и `apps/miniapp` (интерфейс водителя) разделены, но используют общие пакеты.
- **Общие пакеты:** Логика вынесена в `packages/shared-types`, `packages/ui`, `packages/api-client` и `packages/constants`.
- **CI/CD:** Деплой на Vercel через GitHub Actions с фильтрацией по путям.
