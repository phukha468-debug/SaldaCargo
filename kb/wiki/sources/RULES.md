# RULES (Правила разработки)
**Тип:** регламент
**Дата добавления:** 2026-04-29
**Источник:** [[raw/RULES.md]]

## Ключевые идеи
- **Строгие табу:** Никаких удалений из БД, никаких `float` для денег, никаких секретов в коде.
- **Протокол работы:** Исчерпывающее использование AI-инструментов, строгое следование таскам.
- **Стандарты:** Обязательный Try/Catch для async, использование только утвержденного стека технологий.
