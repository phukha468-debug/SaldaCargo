# Роль: Механик
**Теги:** #роли #miniapp #сервис #склад

## Суть
Механик в системе SaldaCargo — это ключевой поставщик данных из СТО. Он фиксирует процесс ремонта, фактически затраченное время и списанные запчасти. Механик финансово изолирован: он не видит цен для клиентов и расчет своей ЗП за конкретный наряд (эти данные обрабатываются администратором в WebApp).

## Функционал и экраны
Работа механика в **MiniApp** организована через управление нарядами и таймерами:

### 1. Управление нарядами
- **Очередь и активные задачи:** Механик видит список нарядов в очереди и может взять их в работу.
- **Новый наряд:** Возможность самостоятельно создать наряд, если машина пришла без предварительного согласования. Требуется ввод [[Одометр|одометра]] и описание проблемы.
- **Работы из каталога:** Выбор операций из справочника `work_catalog`. Если нужной работы нет, создается «свободная работа» для последующего ревью админом.

### 2. Логика таймера и учета времени
- **Точность данных:** Для каждой работы запускается таймер. Система поддерживает статусы: старт, пауза (с указанием причины) и завершение.
- **Ручной ввод:** Если механик забыл запустить таймер, возможен ручной ввод времени с пометкой для администратора.
- **Ограничение:** Одновременно может быть запущен только один таймер.

### 3. Склад и запчасти
- **Списание:** Механик фиксирует, какие детали были фактически использованы для ремонта.
- **Заявки на закупку:** Если запчасти нет на складе, механик формирует черновик заявки для администратора.
- **Просмотр остатков:** Режим read-only для проверки наличия деталей на складе.

### 4. Фиксация дефектов
- В процессе ремонта механик может фиксировать сторонние неисправности в «Бэклог дефектов» с обязательным фото и оценкой срочности.

## Взаимодействие с системой
- Все данные передаются со статусом `draft` для последующего утверждения админом.
- При завершении наряда обязателен ввод конечного [[Одометр|одометра]] (для собственных машин автопарка), что обновляет данные в системе и может триггерить регламентное ТО.
- Используется механизм [[Индемпотентность и Офлайн]] для работы в условиях гаража с плохой связью.
