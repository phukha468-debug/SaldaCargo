## Ключевые правила стека

### Деньги — только строкой

```typescript
// ✅ Правильно
const amount: string = "5000.00";
<Money amount={amount} />

// ❌ Неправильно
const amount: number = 5000;
```

### Supabase: всегда Admin Client

```typescript
import { createAdminClient } from '@/lib/supabase/admin';
// Используется во ВСЕХ API-роутах MiniApp
```

### `as any` на `.from()`

```typescript
// ✅ Правильно
const { data } = await (supabase.from('trips') as any).update({...}).eq('id', id);

// ❌ Неправильно — TypeScript проверит аргументы update()
const { data } = await supabase.from('trips').update({...}).eq('id', id) as any;
```

### TanStack Query — паттерны

```typescript
// Обязательно для форм после мутации:
await queryClient.invalidateQueries({ queryKey: ['trip', id] });
// Потом навигация:
router.push('/driver');

// Для live-данных:
queryFn: ..., staleTime: 0,  // всегда fetch при монтировании
// Для списков:
staleTime: 15000, refetchInterval: 30000,
```
