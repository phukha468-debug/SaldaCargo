# Технический стек
**Теги:** #технологии #разработка #инфраструктура

## Инфраструктура
- **База данных:** Supabase PostgreSQL + RLS + pg_cron.
- **Хостинг:** Vercel (Hobby).
- **Авторизация:** Supabase Auth + MAX OAuth.
- **Хранилище:** Supabase Storage.

## Frontend
- **Framework:** Next.js 15 (App Router), React 19.
- **Стилизация:** Tailwind CSS, shadcn/ui.
- **Графики:** Tremor.
- **Формы и валидация:** react-hook-form, Zod.
- **Управление состоянием и запросы:** TanStack Query.

## Основные правила
- **Деньги:** Только `DECIMAL(12,2)` в БД и `string` в TypeScript для точности.
- **Язык:** UI и документация — русский; код и БД — английский.
- **Безопасность:** Row Level Security (RLS) включен для всех таблиц БД. Проверки прав на уровне приложения вторичны по отношению к политикам БД.
