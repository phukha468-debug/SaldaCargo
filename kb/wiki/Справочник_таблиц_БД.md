# Справочник таблиц БД
**Теги:** #бд #sql #схема #разработка

Этот документ содержит полный перечень таблиц базы данных SaldaCargo и их назначение. Система использует PostgreSQL в инфраструктуре Supabase.

## 1. Ядро системы (Core)
| Таблица | Описание |
| :--- | :--- |
| `legal_entities` | Юридические лица (ИП, ООО). |
| `business_units` | Направления бизнеса (Логистика, СТО, Стоянка). |
| `users` | Персонал и их роли. Привязаны к MAX OAuth ID. |
| `wallets` | Кошельки (кассы, р/с, подотчеты водителей). |
| `categories` | Категории доходов и расходов (дерево). |

## 2. Автопарк (Assets)
| Таблица | Описание |
| :--- | :--- |
| `asset_types` | Типы машин (Газель, Валдай) и их нормы расхода. |
| `assets` | Конкретные автомобили. Хранят одометр и данные об амортизации. |
| `fixed_assets` | Основные средства (оборудование, мебель). |
| `tools` | Инструменты в гараже. |

## 3. Логистика и Рейсы
| Таблица | Описание |
| :--- | :--- |
| `trips` | Путевые листы (рейсы). Связывают машину и водителя. |
| `trip_orders` | Заказы внутри рейса. Хранят сумму и **ручной ввод ЗП**. |
| `trip_expenses` | Расходы, зафиксированные в рейсе (чеки, ГСМ). |

## 4. Финансы (Transactions)
| Таблица | Описание |
| :--- | :--- |
| `transactions` | **Источник истины.** Все движения денег между кошельками. |
| `counterparties` | Клиенты и поставщики. |
| `payroll_periods` | История начислений и выплат ЗП сотрудникам. |

## 5. Сервис и СТО (Maintenance)
| Таблица | Описание |
| :--- | :--- |
| `service_orders` | Заказ-наряды на ремонт. |
| `work_catalog` | Справочник работ с нормами времени и ценами. |
| `service_order_works` | Конкретные работы в наряде с таймерами. |
| `work_time_logs` | Детальный лог работы механика (старт/пауза/стоп). |
| `defect_log` | Бэклог найденных неисправностей. |
| `parts` | Склад запчастей (остатки, цены, пороги). |
| `part_movements` | История движения запчастей (приход/расход). |

## 6. Интеграции
| Таблица | Описание |
| :--- | :--- |
| `fuel_cards` | Привязка топливных карт к машинам. |
| `fuel_transactions_raw` | Сырые данные заправок из Опти24. |
| `bank_statements_raw` | Выписки из банка для последующего матчинга. |
