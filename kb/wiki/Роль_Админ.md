# Роль: Администратор
**Теги:** #роли #администрирование

## Ответственность
Администратор отвечает за управление доступом к системе и контроль за актуальностью данных.

## Управление доступом (MAX Auth)
В системе SaldaCargo доступ реализован через верификацию номера телефона в мессенджере МАКС.

### Как добавить нового пользователя
1.  **Регистрация в БД:** Внесите номер телефона сотрудника в таблицу `users` (формат `79001112233`). Без этого шага вход будет невозможен.
2.  **Первый контакт:** Сотрудник находит бота SaldaCargo в мессенджере МАКС и пишет ему любое сообщение (например, "Привет").
3.  **Верификация в МАКС (Критический шаг):** 
    *   Бот в ответ пришлет сообщение с кнопкой **"Поделиться контактом"**.
    *   Сотрудник должен нажать на эту кнопку.
    *   Мессенджер МАКС покажет системное окно с вопросом: "Поделиться номером телефона с ботом?".
    *   Сотрудник нажимает **"Да/Подтвердить"**.
4.  **Авторизация:** МАКС передаст зашифрованные данные номера на наш сервер. Если номер совпадает с тем, что вы ввели в п.1, сервер выдаст доступ.
5.  **Вход в приложение:** После успешной верификации бот пришлет кнопку **"Открыть приложение"**.

### Как лишить доступа (Увольнение)
Поскольку в системе используется **Soft-delete** (см. [[wiki/Правила_разработки]]), физическое удаление из БД не рекомендуется.
1.  **Блокировка:** Установите пользователю статус, запрещающий вход (например, `is_active = false`) или удалите его роли.
2.  **Проверка:** API авторизации при попытке входа проверяет наличие пользователя. Если запись не найдена или неактивна — вход будет отклонен (ошибка 403).
