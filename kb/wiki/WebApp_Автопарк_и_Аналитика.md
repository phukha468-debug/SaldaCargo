# WebApp: Автопарк и Аналитика

**Теги:** #webapp #аналитика #активы #юнит-экономика  
**Статус:** ✅ РЕАЛИЗОВАНО (09.05.2026)

## Суть

Страница `/fleet` — анализ эффективности каждой единицы техники. Главная цель — выявить убыточные машины и принять решение о продаже.

---

## Реализовано: `/fleet`

### UI: компактные коллапсируемые плитки

- Grid 3 колонки (`grid-cols-1 sm:grid-cols-2 xl:grid-cols-3`), обязательно `items-start`
  - Без `items-start` все карточки в строке растягиваются до высоты открытой — баг раскрытия соседних
- Цветная левая полоса: зелёная (прибыльная), красная (убыточная), жёлтая (ремонт), серая (резерв)
- Фильтр по типу: Все / Грузовики / Газели

**Заголовок тайла (всегда виден):**

- Эмодзи + название + госномер + статус-бейдж
- Количество рейсов за период
- Реальная прибыль (`true_profit`) — после всех расходов, цветная
- Маржа % (зелёная ≥20%, жёлтая 0–20%, красная < 0%)
- Полная себестоимость 1 км (`true_cost_per_km`)
- Мини-бар одометра (красный >80%, жёлтый >55%, зелёный)

**Развёрнутые детали:**

- Красный баннер «Убыточная машина — рассмотреть продажу» если `true_profit < 0`
- Водитель, тип, год, GPS-метка
- Полная разбивка расходов:
  - Операционные (ЗП водителя + грузчика + trip_expenses)
  - Обслуживание (запчасти из нарядов СТО — `part_movements` по выполненным нарядам)
  - Постоянные (monthly_fixed_cost × кол-во месяцев периода)
- Два уровня прибыли:
  - `profit` — операционная (без ТО и постоянных)
  - `true_profit` — реальная (все расходы)
- Доп. метрики: кол-во заказов, средний чек, средний пробег на рейс
- Балансовая стоимость и оставшиеся месяцы амортизации
- Двухшаговое удаление с таймером 4 сек (аналог review/page.tsx)

## API: `GET /api/fleet`

**Параметр:** `?period=current_month|last_month|quarter`

**9 параллельных запросов:**

1. `assets` — все незаархивированные машины с типом и водителем
2. `asset_types` — список типов для формы
3. `trip_orders` — выручка (approved + completed)
4. `trip_expenses` — расходы рейсов (ГСМ и т.д.)
5. `trips` — километры за период
6. `trip_orders` — ЗП водителей и грузчиков
7. `trips` — количество рейсов
8. `trip_orders` — количество заказов (для среднего чека)
9. `part_movements` → `service_orders` — запчасти из завершённых нарядов СТО (`machine_type='own'`)

**Возвращаемая аналитика на машину:**

| Поле               | Описание                       |
| ------------------ | ------------------------------ |
| `revenue`          | Выручка за период              |
| `expenses`         | Операционные (ЗП + ГСМ)        |
| `maintenance`      | Запчасти из СТО                |
| `fixed_cost`       | monthly_fixed_cost × months    |
| `total_costs`      | Сумма всех расходов            |
| `profit`           | Операционная прибыль           |
| `true_profit`      | Реальная прибыль (все расходы) |
| `margin_pct`       | Маржа %                        |
| `km`               | Пробег за период               |
| `trip_count`       | Кол-во рейсов                  |
| `order_count`      | Кол-во заказов                 |
| `avg_order_value`  | Средний чек                    |
| `avg_km_per_trip`  | Средний пробег на рейс         |
| `cost_per_km`      | Операционная себестоимость/км  |
| `true_cost_per_km` | Полная себестоимость/км        |

## API: `POST /api/fleet` / `PATCH /api/fleet/[id]` / `DELETE /api/fleet/[id]`

- **POST** — создание машины (поля: short_name, reg_number, asset_type_id, year, status, odometer_current, assigned_driver_id, current_book_value, remaining_depreciation_months, monthly_fixed_cost, notes)
- **PATCH** — обновление любых полей из разрешённого списка (включая monthly_fixed_cost)
- **DELETE** — физическое удаление только если у машины 0 рейсов; иначе 409 с подсказкой перевести в "Списана"

## Поле `monthly_fixed_cost`

Добавлено миграцией `20260509000003_assets_monthly_fixed_cost.sql`.

```sql
ALTER TABLE assets
  ADD COLUMN IF NOT EXISTS monthly_fixed_cost DECIMAL(12,2) NOT NULL DEFAULT 0;
```

Заполняется вручную через форму редактирования машины. Рекомендуемый расчёт:

- ОСАГО годовой / 12
- Транспортный налог годовой / 12
- Ежемесячный платёж по лизингу/кредиту
