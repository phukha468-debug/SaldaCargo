# Бриф проекта
**Теги:** #бизнес #стратегия #mvp #цели

SaldaCargo — это комплексная система управления транспортным бизнесом, разработанная специально для условий г. Верхняя Салда. Проект направлен на полную оцифровку операционной деятельности ИП Нигамедьянов А.С.

## 1. Основные цели
- **Оцифровка:** Переход от бумажных путевых листов и Excel к единой базе данных.
- **Контроль:** Прозрачность финансовых потоков (движение наличных, подотчеты, дебиторка).
- **Прибыльность:** Достижение целевого показателя чистой прибыли в 1,5 млн руб./мес.
- **Масштабируемость:** Подготовка базы для роста автопарка и направлений бизнеса.

## 2. Ролевая модель (Scope MVP)
- **Владелец:** Стратегический контроль и аналитика.
- **Администратор:** Операционное ревью смен, подтверждение финансовых проводок.
- **Водитель:** Ввод данных о рейсах и расходах через MiniApp в мессенджере MAX.
- **Грузчик:** Просмотр начисленной заработной платы.
- **Механик:** Учет ремонтов, таймеры работ и склад запчастей (следующая фаза).

## 3. Ключевые функции MVP
1. **Электронные путевые листы:** Рейсы, заказы, расчеты в реальном времени.
2. **Учет ЗП:** Ручной ввод вознаграждения водителя/грузчика по каждому заказу (система дает лишь рекомендации).
3. **Финансовый дашборд:** P&L по направлениям бизнеса, балансы всех кошельков.
4. **Управление автопарком:** Амортизация по остаточной стоимости, контроль ТО по пробегу.
5. **Интеграции:** Синхронизация с топливными картами Опти24 и GPS Wialon.

## 4. Контекст бизнеса
- **Локация:** Верхняя Салда.
- **Автопарк:** ~12 единиц техники (Валдаи, Газели, 5-тонник).
- **Команда:** ~15 человек (водители, механики).
- **Юрлицо:** ИП на патенте.
