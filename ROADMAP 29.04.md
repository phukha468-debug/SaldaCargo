## 📌 Заметки для программиста

### Seed-скрипт (TASK_09)

Скрипт `scripts/generate-seed-from-md.ts` должен корректно обрабатывать
строки с `TODO` значениями:

- `max_user_id = TODO` → вставить пользователя с `is_active = false`
- `reg_number = TODO_*` → вставить машину с флагом `needs_update = true`
- Любые другие `TODO` → вставить `NULL`

### МАХ OAuth (TASK_10)

Получить `max_user_id` можно только после того как сотрудник первый раз
откроет бота в МАХ. Процесс:

1. Владелец приезжает на место
2. Каждый сотрудник открывает бота
3. Бот логирует `max_user_id`
4. Владелец переносит ID в `docs/business/people.md`
5. Запускается seed-скрипт → пользователи активируются

### Два водителя «Вова»

В системе два разных водителя с именем Вова:

- Вова на Валдае У446ВН96 (самосвал)
- Вова на Газели 866

В UI отображать как «Вова (Валдай)» и «Вова (Газель 866)».
В БД различаются по `id` и `assigned_asset_id`.

### Кантер 661

Сейчас за рулём Владелец — временно. Машина должна
числиться как активная. Водитель — `owner`.

### Газель 223

Статус `repair`. В UI в разделе Автопарк отображать
отдельно с пометкой «Инвестиционный проект».
Не включать в список доступных машин для водителей.
