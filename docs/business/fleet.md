## Правила для seed-скрипта

При парсинге этого файла скрипт должен:

1. Пропускать строки где `reg_number` начинается с `TODO_` — вставлять с placeholder госномером и флагом `needs_update = true`.
2. Строки с `wialon_id = TODO` — вставлять `NULL`.
3. Строки с `status = repair` — статус `repair`, не `active`.
4. `asset_type_code` → JOIN с таблицей `asset_types` по полю `code`.
